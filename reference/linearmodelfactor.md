# A simulated with linear model factor

A simulated with linear model factor

## Usage

``` r
data(linearmodelfactor)
```

## Format

The variables for `linearmodelfactor` are

- `X`:

  A numeric.

- `Y`:

  A numeric.

- `Z`:

  A factor.
