# Switzerland general liability triangles

`swtri1auto` is a named list of two triangles : the incurred
(cumulative) amounts and the paid (cumulative) amounts.

## Usage

``` r
data(swtri1auto)
```

## Format

`swtriangles` is a named list of two matrices, respectively for incurred
and paid amounts.
